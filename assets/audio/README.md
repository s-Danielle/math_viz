# Audio Assets

## Background music

`solarflex-space-541545.mp3` — the looping ambient background track.
Replace with any track (`.mp3` or `.wav`) and update `backgroundMusicAsset`
in `lib/core/audio/sound_effect.dart` plus the asset path in
`lib/core/audio/audio_service.dart` if the filename changes.

Suggested CC0 sources:
- [HoliznaCC0 on Free Music Archive](https://freemusicarchive.org/music/holiznacc0/)
- [OpenGameArt ambient music](https://opengameart.org/art-search-advanced?keys=&field_art_type_tid%5B%5D=12&sort_by=count&sort_order=DESC)
